Список логів Очистити логи

Документація FastAPI

Огляд

Цей API надає можливості для реєстрації користувачів, входу в систему, перегляду списку всіх користувачів, а також очищення логів.

GET /

Головна сторінка API.

Відповідь: HTML-сторінка з вмістом з файлу home.html.

POST /register

Реєстрація нового користувача.

Тіло запиту (JSON):

{
    "name": "example_user",
    "password": "ExamplePass123!"
}

Відповідь:

  • Успішна реєстрація: 
    {
    "status": "success",
    "username": "example_user",
    "regist_date": "18-04-2025_12:48"
}
  • Помилка 409: Даний юзернейм вже використовується.
  • Помилка 422: Невалідний формат даних (наприклад, пароль не відповідає вимогам).

POST /login

Вхід користувача до системи.

Тіло запиту (JSON):

{
    "name": "example_user",
    "password": "ExamplePass123!"
}

Відповідь:

  • Успішний вхід: 
    {
    "status": "success"
}
  • Помилка 401: Пароль не вірний.
  • Помилка 409: Даний юзернейм не використовується.
  • Помилка 422: Невалідний формат даних.

GET /alluser

Отримання списку всіх користувачів.

Відповідь: JSON-об'єкт з інформацією про користувачів.

GET /log

Отримання логів сервера.

Відповідь: HTML-сторінка з відображенням логів.

GET /log/clear

Очищення логів сервера.

Відповідь: HTML-сторінка з підтвердженням очищення логів.

GET /freerobux

Жартівлива сторінка.

Відповідь: HTML-сторінка з вмістом з файлу fbi/no_virus.html.
